# Tertanum
Mindustry mod предназначен только для экспериментального на данный момент мода.
Мод на данный момент используется только для личного использования и компиляции. У мода нет текстур но он может послужить как обучающий материал для новичков (наподобие мне). 
# Примечание 
Кажется есть спрайтер
# Планы
  1. Полноценная планета с сюжетом.
  2. Большое древо технологий.
  3. Новые вкладки и диалоги.
  4. Новые типы буров, ядер и юнитов.
  5. Реализация межпланетного перелёта.
